## Summary

A minimum borrower contribution from personal funds is required for purchase transactions. The amount varies by occupancy type, LTV/CLTV, number of units, and program (standard, HomeReady, Home Possible). When the contribution requirement is zero, the entire transaction may be funded by eligible gift funds. Investment property transactions require all funds from the borrower's personal funds.

## Key Rules

### FNMA — Standard Loans (Non-HomeReady)

| LTV/CLTV or HCLTV | Property Type | Minimum Contribution from Borrower's Own Funds |
|-------------------|--------------|-----------------------------------------------|
| 80% or less | 1–4 unit primary residence and second home | None — all funds can come from a gift |
| Greater than 80% | 1-unit primary residence | None — all funds can come from a gift |
| Greater than 80% | 2–4 unit primary residence and second home | **5%** minimum from borrower's own funds; gifts can supplement after that |

**Exception — qualifying gift treated as own funds**: If the borrower receives a gift from a relative or domestic partner who has **lived with the borrower for the last 12 months**, or from a **fiancé/fiancée**, the gift is treated as the borrower's own funds and may satisfy the minimum contribution requirement — provided both individuals will use the home as their principal residence. (Source: [[source-loandepot-conventional-guide-2026-03]])

### FNMA — HomeReady Loans

| Property Type | LTV | Minimum Contribution from Borrower's Own Funds |
|--------------|-----|-----------------------------------------------|
| 1–4 unit primary residence | 80% or less | None |
| 1-unit primary residence | Greater than 80% | None |
| 2–4 unit primary residence | Greater than 80% | **3%** of value |

(Source: [[source-loandepot-conventional-guide-2026-03]])

### FHLMC — Standard Loans (Non-Home Possible)

| Property Type / Occupancy | LTV/TLTV/HTLTV ≤ 80% | LTV/TLTV/HTLTV > 80% |
|--------------------------|----------------------|----------------------|
| 1–4 unit primary residence | None | None |
| Second home | None | **5%** of value (when gift funds or grants are used) |
| Investment property | All funds must be from borrower's personal funds | All funds must be from borrower's personal funds |
| Mortgages with shared equity plans | **5%** of value (owner-occupant's personal funds) | **5%** of value |

(Source: [[source-loandepot-conventional-guide-2026-03]])

### FHLMC — Home Possible Loans

| Property Type | LTV/TLTV/HTLTV ≤ 80% | LTV/TLTV/HTLTV > 80% ≤ 95% | LTV/TLTV/HTLTV > 95% |
|--------------|----------------------|-----------------------------|----------------------|
| 1-Unit | None | None | None |
| Manufactured Home | None | None | N/A |
| 2–4 Unit | None | **3%** of value | **3%** of value |

(Source: [[source-loandepot-conventional-guide-2026-03]])

## Exceptions & Nuances

- For **investment properties** (FHLMC standard): all funds must come from the borrower's personal funds — no gifts, no grants.
- Gift of equity is subject to the same minimum borrower contribution requirements as personal gifts — see [[gift-of-equity]].
- The 5% second home threshold (FHLMC, >80% LTV) is triggered specifically when gift funds or grants are used in the transaction; without gifts/grants, the minimum is zero.
